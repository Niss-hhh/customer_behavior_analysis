# 📊 Customer Behavior Analysis  

🚀 **End-to-End Data Analytics Project | SQL • Python • Power BI**

---

## 📌 Project Overview  
This project analyzes customer shopping behavior to uncover actionable insights that drive **data-driven business decisions**, improve **marketing strategies**, and enhance **customer engagement**.  

It demonstrates a complete analytics workflow — from **data cleaning and transformation** to **analysis and dashboard visualization**.

---

## 🎯 Business Objectives  

- Analyze customer purchasing behavior  
- Identify key trends and revenue drivers  
- Segment customers for targeted strategies  
- Improve decision-making using data insights  

---

## 🛠️ Tech Stack  

- Python (Pandas) - Data cleaning & preprocessing  
- SQL (PostgreSQL) - Data querying & analysis  
- Power BI - Interactive dashboard & visualization

## 🔍 Key Dashboard Metrics  

- 👥 **Total Customers:** 3.9K  
- ⭐ **Average Rating:** 3.75  
- 💰 **Average Purchase Value:** $59.76  

---

## 📊 Key Insights (Business Impact)  

- 💰 **Clothing category generates the highest revenue**, making it the primary revenue driver  
- 🛍️ **Accessories and footwear show moderate performance**, indicating upsell opportunities  
- ⭐ **Average rating (3.75) suggests scope for improving customer satisfaction**  
- 🔁 **Only 27% customers are subscribed**, highlighting a major growth opportunity  
- 🚚 **Shipping preferences vary**, suggesting potential for personalized delivery strategies  

---

## 📈 Dashboard Features  

- Interactive filters (Subscription Status, Gender, Category, Shipping Type)  
- Category-wise sales and revenue breakdown  
- Customer segmentation insights  
- KPI cards for quick business understanding
